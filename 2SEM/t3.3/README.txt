ЦЫБУЛИН ЕГОР 108 группа (2025 год)


Условие: Заменить всюду в файле один заданный набор символов на другой (с учётом разницы в их длине).


Входные данные: input.txt — файл с символьными строками


Выходные данные: output.txt — файл с обработанными строками.


Проверка производилась: Linux Ubuntu 22.04.3 LTS AMD64 (WSL 2 for Windows 11) + gcc 11.4.0


Запуск программы:
gcc t3.3.c
./a.out


Общее описание: Внутри программы в #define можно поменять максимальный размер строки. Программа отрабатывает случай, когда входная строка длиннее допустимого значения. В таком случае она прерывает работу программы, и именно поэтому предотвращены сбои внутри самой функции.
