
Пример кода ввода и обработки массива.

Исходные данные записываются в файле --- несколько чисел.
При запуске имя файла данных вводится с клавиатуры. 

Функции для обслуживания массива вынесены в отдельный файл array.c
а их прототипы - в файл array.h

В коде делается много проверок корректности.

Компиляция: 
gсс minmax2.c test-minmax.c array.c -o prog

Запуск:
prog
