ЦЫБУЛИН ЕГОР 108 группа (2025 год)


Условие: Написать программу, которая по аргументам x_1, ..., x_n и значениям функции y_1, ..., y_n вычисляет приближённые значения данной функции на заданной сетке \bar{x_1}, ..., \bar{x_n} с помощью:
1. Интерполяционного многочлена Лагранжа;
2. Кусочно-линейной интерполяции;
3. Кусочно-квадратичной интерполяции;
4. Эрмитовой интерполяции;
5. Наилучшего среднеквадратического приближения линейной функцией.


Входные данные: В input.txt по столбцам аргумент, значение функции от аргумента, значение производной от аргумента; В grid.txt — новые точки сетки.

Выходные данные: Программа выводит в консоль сообщение об успешном завершении работы "success", а в выходные файлы выводит аргумент из новой сетки и приближённое значение в этой точке.

Проверка производилась: Linux Ubuntu 22.04.3 LTS AMD64 (WSL 2 for Windows 11) + gcc 11.4.0


Запуск программы:
gcc generator.c
./a.out
gcc fun.c main.c
./a.out


Замечания: В составе программы есть генератор точек для input.txt и grid.txt: там можно написать функцию, с которой в будущем будем работать. Для интереса там есть смещение координат новых точек, а также изменено их количество.
