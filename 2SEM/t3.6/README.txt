ЦЫБУЛИН ЕГОР 108 группа (2025 год)


Условие: Вывести все слова из данного файла в алфавитном порядке без повторений.


Входные данные: input.txt — файл с символьными строками


Выходные данные: В консоль выводит необходимую диагностику и отсортированный словарь.


Проверка производилась: Linux Ubuntu 22.04.3 LTS AMD64 (WSL 2 for Windows 11) + gcc 11.4.0


Запуск программы:
gcc t3.4.c
./a.out


Общее описание: Внутри программы в #define можно поменять максимальный размер строки. Программа выводит всю необходимую диагностику: если слово длиннее максимального размера, то оно далее не учитывается и в словарь не вносится даже в частичном варианте. Есть два сценария: 1 — сортировка с учётом регистра; 2 — сортировка без учёта регистра.
