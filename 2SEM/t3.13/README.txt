Условие: Реализовать инструкции типа #ifdef - #else - #endif, т.е. оставить в файле требуемый текст в зависимости от условия.


Входные данные: input.c — файл с символьными строками (поменять название / расширение файла можно в функции main).


Выходные данные: В консоль выводит диагностику: или сообщение об ошибке, или сообщение об успешном завершении работы программы. В файл "output.c" (поменять название / расширение файла можно в функции main) выводит результат работы программы.


Проверка производилась: Linux Ubuntu 22.04.3 LTS AMD64 (WSL 2 for Windows 11) + gcc 11.4.0


Запуск программы:
gcc t3.13.c
./a.out


Общее описание: Внутри программы в #define можно поменять максимальный размер строки и прочие параметры. Программа отрабатывает случаи выхода за допустимые размеры — в таком случае она безопасно прерывает работу программы.
