Формулировка задания: Получить массив биномиальных коэффициентов для степени N, последовательно вычисляя строки треугольника Паскаля (используя только один массив)

ЦЫБУЛИН ЕГОР, МЕХМАТ МГУ, 108 ГРУППА, GitHub: https://github.com/betel-git/1-sem-msu-c

Версия: 1.0 (21.10.2024)

Состав программы: 2.18.c, fun.c, fun.h, readme.txt
Формат входных данных: Число N от -32767 до +32767
Формат выходных данных: Последовательность найдённых биномиальных коэффициентов "output.txt"

Код компилировался на дистрибутиве Ubuntu 22.04.3 LTS (Linux 5.15.153.1-microsoft-standard-WSL2) и gcc 11.4.0 компиляторе с максимальными ключами проверки.
Также проверка производилась на ОС Windows 11 10.0.22631 (amd_x64) и gcc 10.2.0 компиляторе с максимальными ключами проверки.
Сборка кода: sh wgcc 2.18.c fun.c -o prod - на Linux.
Запуск кода: ./prod - на Linux.
