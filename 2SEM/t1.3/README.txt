ЦЫБУЛИН ЕГОР 108 группа (2025 год)


Условие: Вычислить минимальное значение данной функции f(x) с заданной точностью eps методом построения параболы с поиском трёх точек с помощью:
1. Измельчения геометрической прогрессии;
2. Метода золотого сечения;
3. Измельчения равномерной сетки.


Входные данные: Внутри main.c можно изменить отрезок, на котором ищем минимум, и точность eps. Функции можно поменять в tests.c

Выходные данные: Программа выводит в консоль название метода, которым находился минимум, номер теста, успешность его прохождения, количество итераций, значение x, значение f(x), длина плато (если найдено), сообщение о возможном наличии плато.

Проверка производилась: Linux Ubuntu 22.04.3 LTS AMD64 (WSL 2 for Windows 11) + gcc 11.4.0


Запуск программы:
gcc ver3.c main.c tests.c -lm
./a.out


Замечания: Функция из примера с занятий действительно выдаёт максимальную точность 1e-5. Присутствует ограничение на количество итераций.

Замечания: Метод золотого сечения быстрее дойдёт до деления нуля, чем до выявления плато.
