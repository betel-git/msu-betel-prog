ЦЫБУЛИН ЕГОР 108 группа (2025 год)


Условие: Определить количество, максимальную, минимальную и среднюю длину слов из данного файла, а также частоту (процент) появления каждого символа.


Входные данные: input.c — файл с символьными строками (поменять название / расширение файла можно в функции main).


Выходные данные: В консоль выводит диагностику: или сообщение об ошибке, или сообщение об успешном завершении работы программы, а также результаты работы программы: количество слов; максимальную, минимальную, среднюю длины слов; частоту встречи символов в процентах.


Проверка производилась: Linux Ubuntu 22.04.3 LTS AMD64 (WSL 2 for Windows 11) + gcc 11.4.0


Запуск программы:
gcc t3.5.c
./a.out


Общее описание: Внутри программы в #define можно поменять максимальный размер слова. Программа отрабатывает случаи выхода за допустимые размеры — в таком случае она безопасно прерывает работу программы.
