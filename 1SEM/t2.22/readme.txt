Код компилировался на дистрибутиве Ubuntu 22.04.3 LTS (Linux 5.15.153.1-microsoft-standard-WSL2) и gcc 11.4.0 компиляторе с максимальными ключами проверки.
Проверка производилась на файлах input1.txt и input2.txt (формат: каждое число в отдельной строке, в конце пустая строка).
Результат программа выводит в файл output1.txt и output2.txt.
Также проверка производилась на ОС Windows 11 10.0.22631 (amd_x64) и gcc 10.2.0 компиляторе с максимальными ключами проверки.
Запуск кода: sh wgcc 18.c fun.c -c && sh wgcc 18.o fun.o -o prod && ./prod - на Linux.
