ЦЫБУЛИН ЕГОР 108 группа (2025 год)

Условие: Вычислить корень уравнения на данном отрезке с заданной точностью методом Ньютона.
Условия на функцию: Функция должна быть непрерывной и дифференцируемой на отрезке.
Входные данные: В INPUT.txt значения eps — точность вычислений, a — левая граница отрезка, b — правая граница отрезка по строкам.
Выходные данные: В консоль программа должна выводить сообщение "The results are saved in OUTPUT.txt" или сообщения об ошибке открытия файлов. В OUTPUT.txt хранятся входные значения, номер теста, предупреждения, результат работы функции, количество итераций.
Проверка производилась: Linux Ubuntu 22.04.3 LTS AMD64 (WSL 2 for Windows 11) + gcc 11.4.0

Замечание 1: Программа будет предупреждать, если найденный корень будет вне заданного отрезка, тогда предполагается, что пользователь сам задаст более подходящий отрезок (также можно попробовать поменять метод выбора начальной точки в функции main).
Замечание 2: Если значение eps будет меньше, чем машинная точность, то программа выдаст ошибку (машинная точность для используемого в программе типа данных double — ≈2e-16).
Замечание 3: Основные проблемы возникают в случае, когда производная стремится к нулю при стремлении точки к корню (например, для x**3) или если функция периодическая.

Состав проекта:
1. functions.c — файл с функциями
2. main.c — файл с функцией main
3. headers.h — заголовочный файл
4. INPUT.txt — входной файл
5. OUTPUT.txt — выходной файл
6. README.txt — инструкция

Запуск программы:
gcc functions.c main.c -lm
./a.out

Описание тестов:
1. Если eps >= fabs(b - a)
2. Малое значение eps, границы не симметричны относительно нуля
3. Относительно второго теста: увеличение точности
4. Границы симметричны относительно нуля
5. Относительно четвёртого теста: увеличение границ
6. Граница [1.25, 4] - универсальна для почти всех (кроме куба) функций (т.е. функции на этом отрезке имеют ровно 1 корень)
7. Относительно шестого теста: огромное значение eps
8. Нечисловое значение во входном файле.

Правильные ответы:
1. \pm \sqrt{2}
2. 0
3. 0, \pi * k, k \in Z
4. 2.5
5. 1, \frac{1 \pm \sqrt{5}}{2} (1.61, -0.61)
