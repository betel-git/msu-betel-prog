ЦЫБУЛИН ЕГОР 108 группа (2025 год)


Условие: Написать программу вычисления значения элементарных функций (sin, cos, exp, log) в заданной точке x и с заданной точностью суммированием ряда Тейлора. Сравнить с точным ответом.
Входные данные: В INPUT.txt значения eps — точность вычислений, x — точка, в которой считаем значение.
Выходные данные: Программа выводит в консоль сообщение об успешном завершении работы "Success", а в выходные файлы выводит всю диагностику по каждой функции отдельно.
Проверка производилась: Linux Ubuntu 22.04.3 LTS AMD64 (WSL 2 for Windows 11) + gcc 11.4.0


Состав проекта:
1. sin.c, cos.c, exp.c, log.c — файлы с функциями
2. main.c — файл с функцией main
3. headers.h — заголовочный файл
4. insin.txt, incos.txt, inexp.txt, inlog.txt — входной файл
5. README.txt — инструкция

Запуск программы:
gcc sin.c cos.c exp.c log.c main.c -lm
./a.out
